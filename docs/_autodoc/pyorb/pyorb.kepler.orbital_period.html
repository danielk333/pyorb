orbital_period¶

orbital_period

pyorb.kepler.orbital_period(a, mu)[source]¶

Calculates the orbital period of an Keplerian orbit based on the semi-major axis \(P = 2\pi\sqrt{\frac{a^3}{\mu}}\).

Parameters

  • a (float/numpy.ndarray) – Semi-major axis of ellipse.

  • mu (float) – Standard gravitation parameter \(\mu = G(m_1 + m_2)\) of the orbit.

Returns

Orbital period.
