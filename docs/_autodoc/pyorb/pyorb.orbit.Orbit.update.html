Orbit.update¶

Orbit.update

Orbit.update(inds=slice(None, None, None), **kwargs)[source]¶

Calculates Keplerian elements based on Cartesian elements

Note: Cannot update BOTH cartesian and Keplerian elements.

Parameters

inds (int/list/numpy.ndarray/slice) – Incidences of orbits to update, defaults to all

Returns

None

Keyword arguments

  • Any valid Cartesian or Keplerian variable name can be given as a key to update that parameter.

  • Mass can also be updated this way but this requires manual execution of either calculate_cartesian or calculate_kepler depending on which are to remain constant.
