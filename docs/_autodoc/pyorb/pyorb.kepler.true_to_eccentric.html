true_to_eccentric¶

true_to_eccentric

pyorb.kepler.true_to_eccentric(nu, e, degrees=True)[source]¶

Calculates the eccentric anomaly from the true anomaly.

Parameters

  • nu (float/numpy.ndarray) – True anomaly.

  • e (float/numpy.ndarray) – Eccentricity of ellipse.

  • degrees (bool) – If true degrees are used, else all angles are given in radians

Returns

Eccentric anomaly.

Return type

numpy.ndarray or float
