eccentric_to_true¶

eccentric_to_true

pyorb.kepler.eccentric_to_true(E, e, degrees=True)[source]¶

Calculates the true anomaly from the eccentric anomaly.

Parameters

  • E (float/numpy.ndarray) – Eccentric anomaly.

  • e (float/numpy.ndarray) – Eccentricity of ellipse.

  • degrees (bool) – If true degrees are used, else all angles are given in radians

Returns

True anomaly.

Return type

numpy.ndarray or float
