true_to_mean¶

true_to_mean

pyorb.kepler.true_to_mean(nu, e, degrees=False)[source]¶

Transforms true anomaly to mean anomaly.

Uses:

  • true_to_eccentric()

  • eccentric_to_mean()

Parameters

  • nu (float/numpy.ndarray) – True anomaly.

  • e (float/numpy.ndarray) – Eccentricity of ellipse.

  • degrees (bool) – If true degrees are used, else all angles are given in radians

Returns

Mean anomaly.

Return type

numpy.ndarray or float
