semi_major_axis¶

semi_major_axis

pyorb.kepler.semi_major_axis(P, mu)[source]¶

Calculates the orbital semi-major axis of an Keplerian orbit based on the orbital period \(a = \mu^{\frac{1}{3}}(\frac{P}{2\pi})^{\frac{2}{3}}\).

Parameters

  • P (float/numpy.ndarray) – Orbital period

  • mu (float/numpy.ndarray) – Standard gravitation parameter \(\mu = G(m_1 + m_2)\) of the orbit.

Returns

semi-major axis.
